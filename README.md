# Memento

A minimal dashboard for remembering your time.

Inspired by the original [Motivation](https://github.com/maccman/motivation) by Alex MacCaw.  
This version is a full rewrite and does not substantially reuse the original codebase.

## Features

- Replaces the Chrome new tab page
- Live age counter
- Estimated remaining lifetime countdown
- Custom birthday and life expectancy
- Custom page title and favicon
- Optional shortcut titles
- Shortcut click counters
- Custom shortcut icons
- Custom dashboard font
- Local-only settings using browser storage

## Installation

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Choose Load unpacked.
4. Select this project folder.

## Privacy

Memento stores settings locally in the browser. The extension does not require an account and does not send user settings to an external server. See `PRIVACY.md` for details.
