# Memento Privacy Policy

Memento stores user-provided settings locally in the browser. This may include birthday, life expectancy value, display text, uploaded icons or fonts, shortcuts, shortcut order, shortcut icon choices, and optional shortcut click counters.

Memento does not transmit this data to the developer or to any third party. Memento does not sell user data, use user data for advertising, or use user data for creditworthiness or lending purposes.

Shortcut links only navigate to the URLs chosen by the user when clicked. Memento does not read browsing history, website content, emails, messages, location, or authentication data.

Users can delete all stored data at any time using the reset settings option in the extension settings, or by removing the extension from Chrome.
